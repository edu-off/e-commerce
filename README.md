* Para subir tdodos os containeres e bancos de dados da aplicação via Makefile:
```shell
make docker-start
```

<br><br/>

* Para subir container da aplicação e servidor de banco de dados via docker compose:
```shell
docker-compose -f docker-compose.yml up -d
```
